# tetratonicsquares
A music toy built in Python with Tkinter and Pygame.

## Getting Started

```
git clone https://github.com/joemarchese/tetratonicsquares
```

## Prerequisites
Python 3.x
Pygame
```
pip install pygame
```

## How to Play
* Click.
* Click some more.
